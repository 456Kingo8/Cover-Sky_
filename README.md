# 遮天计划
【体系】:轮海，道宫，四极，化龙，仙台
特点:一世成帝一人，战力无双，寿元稀缺。

## 境界
**轮海境**
        
*   **苦海**开辟苦海，容纳生命精气，是修行的起点。
        
*   **命泉**苦海中涌现命泉，生命精气更加旺盛，可以初步运用一些法术。
        
*   **神桥**架起神桥，连接苦海与彼岸，可以飞行。
        
*   **彼岸**抵达彼岸，完成轮海境的修炼，可以尝试突破道宫境。
    
*   **道宫境：** 五大神藏，分别对应五行，每个神藏的修炼都会带来不同的能力提升。
    
*   **四极境：** 四肢修炼到极致，肉身力量大幅提升，可以进行空间穿梭。
    
*   **化龙境：** 脊椎大龙的九次蜕变，每一次蜕变都伴随着实力的巨大提升。

**仙台境：**
        
* **仙一半步大能：** 初步踏入仙台，可以称雄一方。
   
* **仙二大能：** 实力强大，可以开宗立派。

*   **仙三斩道王者：** 斩去杂念，明悟自身之道，是成圣的关键一步。斩道的过程凶险异常，需要斩去过往的执念和羁绊，才能明悟自身的道。斩去的天赋和执念越多，斩道后的实力越强大。可以设立一个“斩道”事件，让玩家选择要斩去的天赋，并根据斩去的天赋数量提升实力。
 
* **仙四圣人：** 超脱凡尘，拥有强大的实力和神通，可以开创圣地。

*   **仙五圣人王：** 圣人中的王者，实力更加强大，可以镇压一方星域。

*   **仙六大圣：** 距离准帝只有一步之遥，可以与准帝争锋。

*   **准帝：** 拥有准帝领域，是帝路争锋的参与者。准帝拥有强大的实力和独特的领域，可以大幅度削弱领域内敌人的实力。准帝的领域对准帝和大帝无效，但对其他境界的修士有显著的压制效果。

* **大帝：** 一个时代只能有一位大帝，拥有无敌的实力和天心印记，镇压万古。大帝拥有帝之领域，对帝境以下的修士有极大的压制效果。大帝的领域对帝境无效，但对其他境界的修士有近乎无敌的压制力。

# 机制
## 功法机制
**1. 功法机制**

*   **功法等级:** 你的功法列表很棒，可以根据功法等级设定不同的修炼速度加成和技能。
*   **功法获取:** 可以通过任务、击杀特定生物、自行领悟等。
*   **功法残页:** 可以设定一些残缺的功法，需要收集齐所有残页才能合成完整的功法。
*   **功法领悟:** 学习高级功法，可以对悟性有要求，或者添加领悟度，领悟度满才能提升等级。
*   **功法搭配:** 不同的功法之间可以存在相生相克的关系，合理的功法搭配可以提升战斗力。
*   **经文共鸣:**  可以设定一些特殊事件，例如持有某本经书的角色在特定地点可以触发经文共鸣，获得顿悟或特殊奖励。

    *   **自动生成功法机制：**
        *   **灵感触发：** 当角色达到一定境界或满足特定条件时，有一定几率触发灵感，自动领悟新的功法。
        *   **推演融合：** 角色可以根据已有的功法进行推演和融合，创造出新的功法。这个过程可以消耗一定的资源或时间，成功率和创造出的功法等级与角色的悟性、已掌握的功法等因素有关。
        *   **天道补全：**  在一些特殊的地点或事件中，可以触发天道补全机制，将一些残缺的功法自动补全。

## 特定机制
*   **准帝突破至尊：** 设定一个“帝路资格”的天赋，只有拥有这个天赋的准帝才能在帝路开启时突破至尊，争夺帝位。
*   **红尘仙:** 寿命终结时，有极低的几率触发红尘仙机制，重活一世，并保留部分记忆和修为。可以通过设定一些条件来增加触发红尘仙的几率，例如服用不死药、修炼特殊功法等。
### 帝路争锋
*   **自动挡:** 系统自动模拟帝路争锋的过程，根据准帝的实力和天赋自动决出胜者。
*   **手动挡:** 玩家可以手动操控自己的角色参与帝路争锋，与其他准帝进行战斗。可以设定一些特殊的战场或规则，增加战斗的策略性和趣味性。
*   **帝路buff:** 参与帝路争锋的准帝都会获得一个buff，提升自己的实力。
*   **天心印记:** 帝路争锋的胜者可以融合天心印记，成为大帝，获得极大的实力提升和天道加持。
*   **极道镇压:** 大帝存活时，会镇压万古，阻止其他人成帝。可以设定一个“极道压制”的debuff，阻止所有角色突破到准帝及以上境界。
#### 黑暗动乱
*   **黑暗纪元开关:** 开启后，地图上会刷新黑化的准帝，他们会攻击其他角色。
*   **黑化特质:** 黑化的准帝会获得一个特殊的“黑暗”特质，提升他们的实力，并使他们具有一些特殊的能力。
*   **黑暗动乱的平定:** 玩家可以通过击杀黑化的准帝来平定黑暗动乱，并获得奖励。可以设计一些特殊的事件或任务，引导玩家参与到黑暗动乱的平定中。
